3.1. GpyFFT¶

class gpyfft.GpyFFT¶

The GpyFFT object is the primary interface to the AMD FFT library

Methods

get_version()¶

returns the version of the underlying AMD FFT library

Returns:

A tuple with the major, minor, and patch level of the AMD FFT library.

example: (1L, 8L, 214L)

Raises:
GpyFFT_Error: An error occurred accessing the clAmdFftGetVersion function
